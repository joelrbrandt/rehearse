Background: Processing

Processing is a programming environment for creating interactive graphics. The udnerlying language is Java. Processing files have a simple structure: on startup, the function setup() is called once, then the function loop() is called repeatedly. So a simple processing program would look like this:

int i;
void setup() {
i=0;
}
void loop() {
println(i++);
}

Processing ships with lots of excellent examples, available through the File->Examples menu in Processing. Online reference is available at http://processing.org/reference/:

Rehearse and HelpMeOut

Rehearse is an augmented editor for Processing. HelpMeOut is (will be) a tool for suggesting code examples while you program.

To enable data gathering, please show the HelpMeOut window via the Tools->Show/Hide HelpMeOut menu; also please only compile and run your programs using the second "run" button in the toolbar (not the "interactive run"). Here's a video that shows these steps:

2. Paint
Make a simple "Paint" program. It should have a toolbar with tools and a canvas on which to draw. There should be at least a "pencil" tool which draws in color on the canvas, and an "eraser" tool which removes the color. Add more tools if you want!

Examples:
Basics > Input > MouseFunctions, MousePress
Topics > GUI > Buttons
Topics > Drawing > ContinuousLines

Here's an example of what your program could/should look like:
http://screencast.com/t/byVaa7cF

3. Spotlight
Create a spotlight which lights a 3D "stage" and follows the cursor around on it. Taking perspective into account, moving the cursor upstage (away from the camera) should make the spotlight get smaller, while moving it downstage (towards the camera) should make it get bigger.

Examples: 3D > Lights > Directional, Spot

Hint: try using "box()" to get a flat surface (see example "Lights1")
